# Multimodal Test Kit

Sample sources to test every input modality of Block Research LM.

| File | Modality | Pipeline tested |
|---|---|---|
| sample.md / sample.txt | text/markdown | parser + chunking + embeddings |
| sample.csv | tabular | text parser |
| sample.html | web/document | html-to-text |
| sample-image.png | image | GPT-5.5 vision OCR/description |
| sample-audio.mp3 | audio | Deepgram Nova-3 transcription |

The image and audio are **AI-generated by the app's own APIs** (gpt-image-2, ElevenLabs),
then fed back as inputs — an end-to-end "output becomes input" multimodal test.
