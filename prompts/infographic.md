# Infographic Prompt

Generate a source-grounded infographic plan and render-ready SVG brief from Evidence Pack items.

Rules:
- Use only Evidence Pack items and Knowledge Layer objects.
- Every panel must carry a citation and source refs.
- Produce layout metadata: orientation, detail level, visual style, dimensions, panels, evidence appendix, and renderer notes.
- Keep panel copy short enough for fixed SVG/card layouts.
- Do not introduce decorative claims, unsupported metrics, or generic model-memory facts.
- Return valid JSON and preserve citations.
